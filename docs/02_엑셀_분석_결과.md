## Schedule 시트 - 핵심 데이터 모델

### WBS 영역 (A~AE 컬럼, 행 5~56)

| 컬럼 | 헤더 | 설명 | 숨김 |
|------|------|------|------|
| A | (의존관계) | "타입;행번호" 예: "1;6" = FS, 행6 | YES |
| B | (그룹마커) | "G" = 그룹/요약 작업 | YES |
| C | (WBS레벨) | 계층 숫자 (1~6) | YES |
| D | WBS | WBS 코드 ("1", "1.1", "1.1.2.1" 등) | No |
| E~N | 작업* | 작업명 (레벨별 들여쓰기, 최대 10단계) | No |
| O | 비고 | 메모/비고 | No |
| P | 시작일* | 시작일 | No |
| Q | 완료일* | 완료일 | No |
| R | Calendar | 적용 달력 (STD, UD1, UD2) | YES |
| S | 총 작업량 | Man/Day | No |
| T | 계획 작업량 | Man/Day | No |
| U | 총 기간 | 일수 | No |
| V | 계획 기간 | 일수 | No |
| W | 실제 시작일 | | YES |
| X | 실제 완료일 | | YES |
| Y | 실제 총작업량 | | YES |
| Z | 실제 총기간 | | YES |
| AA | 자원수 | 리소스 수 | YES |
| AB | 담당 | 리소스 배정 (이름[%], 쉼표구분) | No |
| AC | 산출물 | 산출물명 | No |
| AD | 계획 | 계획 진척률 (0~1) | No |
| AE | 실적* | 실적 진척률 (0~1, 사용자 입력) | No |

### 간트차트 영역 (AF~GJ 컬럼)
- 주간(Weekly) 단위 타임라인
- VBA 매크로가 셀 배경색으로 바 렌더링
- Row 4: 각 컬럼의 월요일 날짜
- Row 2: 프로젝트 주차 번호 (W1, W5, W9...)

### 의존관계 타입 (컬럼 A)
| 코드 | 의미 |
|------|------|
| 1 | FS (Finish-to-Start) |
| 2 | SS (Start-to-Start) |
| 3 | FF (Finish-to-Finish) |
| 4 | SF (Start-to-Finish) |

## AnalysisReport 시트

월별 진척 테이블 (6개 기간: 2025-05-19 ~ 2025-10-31)
- 컬럼: 시작, 종료, 계획작업량, 누적계획, 계획율, EV, 누적EV, EV율, 실적작업량, 누적실적, 실적율

## VBA 핵심 로직 (데이터 플로우)

```
사용자 입력 (작업명, 날짜, 진척률, 담당자)
    │
    ▼
VBA "스케줄업데이트" 매크로
    │
    ├→ 작업량 계산 (자원수 × 배정률)
    ├→ 기간 계산 (비근무일 제외, Calendar 참조)
    ├→ 계획 진척률 계산 (기준일 vs 일정)
    ├→ 간트바 렌더링 (셀 배경색 + 텍스트 수식)
    ├→ Workload 시트 갱신 (플랫 데이터)
    ├→ Progress 시트 갱신 (총계/비율)
    └→ AnalysisReport 갱신 (월별 집계 + 차트)
```
