# XLGantt Web - 프로젝트 개요

## 목적
엑셀 기반 간트차트 도구 **XLGantt v6.0.0** (.xlsm)을 웹 애플리케이션으로 마이그레이션

## 원본 파일
- **파일명:** XLGantt_v6.0.0_20250810_KOR.xlsm
- **제작:** XLWorks (xlworks.net), Freeware
- **참고:** https://xlworks.net/xlgantt-project-scheduler/

## 기술 스택 (확정)
| 항목 | 선택 |
|------|------|
| 프레임워크 | React 19 + TypeScript |
| 빌드 | Vite |
| 상태관리 | Zustand |
| UI | Tailwind CSS + shadcn/ui |
| 차트 | Recharts |
| 날짜 | date-fns |
| DB | Supabase (PostgreSQL) |
| 배포 | Vercel |
| 가상화 | @tanstack/react-virtual |
| 엑셀 파싱 | xlsx (SheetJS) |

## 범위
- 전체 기능 구현 (뷰어가 아닌 편집 가능한 프로젝트 관리 도구)
- 1920x1080 해상도 최적화
