# 사용자 피드백 및 미구현 기능 목록

## 사용자 피드백 이력

### 피드백 1: 초기 구조
- 햄버거 메뉴 불필요 → 탭 네비게이션으로 변경 완료
- 담당자 배정/WBS 추가·수정·순서변경·선행작업 설정 필요 → 툴바 + 상세편집 구현

### 피드백 2: 간트차트
- 마일스톤 표시가 거칠다 → 다이아몬드 SVG 개선
- 담당자 등록해도 담당 컬럼에 표시 안됨 → resource-store 연동

### 피드백 3: 폰트/크기
- 전체 폰트가 너무 작다 → 1920x1080 기준 2배 확대 적용

### 피드백 4: 테이블-간트 정렬
- 테이블 행과 간트차트 행 높이 불일치 (특히 최종보고 등 마지막 행)

### 피드백 5: 작업 상세편집 다이얼로그 (중요)
1. **탭이 너무 많다** → 탭 제거, 단일 스크롤 페이지로 변경 필요
2. **칸이 너무 좁다** → 넓게 리디자인
3. **산출물이 동작 안함** → 파일첨부 기능 (멀티파일) 구현 필요
4. **마일스톤** → 체크박스 하나면 충분
5. **담당자 추가 후 "mem-002"로 표시됨** → ID 대신 이름 표시 필요
6. **선행작업 추가 시 난수로 보임** → 작업명으로 표시 필요
7. **세부항목 추가해도 리스트에 반영 안됨** → 표시 로직 수정 필요
8. **세부작업항목 개념** → 서브태스크 + 체크리스트 둘 다 지원

### 피드백 6: 진척현황
1. 수치 표시의 필요성 재검토
2. 작업별 진척을 보여주는 이유가 불명확
3. 기준일자 지정 기능 없음 (금일 기준만)

### 피드백 7: 달력
1. 현재 공휴일 등록만 있음 → 이건 "관리" 기능
2. 달력은 간트와 다른 캘린더 뷰 형태로 보여줘야 함 → 월간 캘린더 뷰 필요

### 피드백 8: 담당자
1. 등록만 있고 수정이 없음
2. 회사별 색상도 수정 불가

## 엑셀 대비 미구현 기능 도출

### A. Schedule 시트 관련
| 엑셀 기능 | 웹 구현 상태 | 우선순위 |
|-----------|-------------|---------|
| 실제 시작일/완료일 (W, X열) | ❌ 미구현 | 높음 |
| 실제 작업량/기간 (Y, Z열) | ❌ 미구현 | 높음 |
| 자원수 자동계산 (AA열) | ❌ 미구현 | 중간 |
| 비고 필드 (O열) | ❌ 미구현 | 낮음 |
| 달력 선택 (STD/UD1/UD2) per task (R열) | ❌ 미구현 | 중간 |
| 계획 작업량 vs 총 작업량 구분 (S, T열) | ❌ 부분 구현 | 중간 |
| 계획 기간 vs 총 기간 구분 (U, V열) | ❌ 부분 구현 | 중간 |
| 간트바 텍스트 라벨 (진척% + 작업명 + 담당 + 기간) | ❌ 부분 구현 | 높음 |
| 그룹 작업 자동 롤업 (날짜, 작업량, 기간) | ❌ 부분 구현 | 높음 |
| 타임스케일 줌 (주간/월간/일간) | ❌ 주간만 | 중간 |
| 진행선 (Progress Line) | ❌ 미구현 | 중간 |
